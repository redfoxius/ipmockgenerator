# ipmockgenerator

Usage of ipmockgenerator:

````
-mask [string]
    IP mask for gen, a string. Default: '212.121.*.*' (default "212.121.*.*")
-num [int]
    number of IPs in file, an int. Default: 10000 (default 1000)
-out [string]
    output file name, a string. Default: ip.txt (default "ip.txt")
